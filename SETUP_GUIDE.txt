ORION AI - Quick Setup Guide
=============================

STEP 1: Ensure MySQL is Running
- Start XAMPP Control Panel
- Start Apache and MySQL services

STEP 2: Database is Already Created
- Database name: OrionLearn
- Already migrated and seeded with sample data

STEP 3: Start the Application
Run this command in the orion-ai directory:
    php artisan serve

STEP 4: Access the Website
- Main Website: http://localhost:8000
- Admin Login: http://localhost:8000/login

NOTES:
======
- Email notifications are set to 'log' mode (check storage/logs/laravel.log)
- For production, configure SMTP in .env file
- File uploads go to storage/app/public/
- Run 'php artisan storage:link' if uploads don't work

TROUBLESHOOTING:
================
If you get errors:
1. Clear cache: php artisan cache:clear
2. Clear config: php artisan config:clear
3. Recreate database: php artisan migrate:fresh --seed
4. Check MySQL is running in XAMPP
