# ORION AI Logo Files

This directory contains the logo and favicon files for the ORION AI platform.

## Files:
- `logo.png` - Main logo file (used in navigation and as favicon)
- `favicon.png` - Favicon file (browser tab icon)

## Source:
The original logo is stored in `storage/app/public/logo/orion-logo.png` and is accessible via the storage link at `/storage/logo/orion-logo.png`.

These public copies ensure the logo is available even before the storage link is created during deployment.
